# Classic Decoder App V2.0.2 - IAP Test Suite

## Description
Manual QA documentation and Build Verification Suite for **Classic Decoder App V2.0.2**. This repository validates the transition from Stripe/PayPal to Google Play In-App Purchases (IAP) to ensure compliance with Play Store policies. Includes test cases, execution checklists, and final QA reports.

## Project Scope
- **Payment Migration:** Verified the transition from external gateways (Stripe/PayPal) to Google Play Billing.
- **User Authentication:** Verified Signup, Login, and Profile persistence.
- **Content Generation:** Validated Report & Sticker generation and high-fidelity preview pages.
- **Cross-Platform Compatibility:** Tested on Android versions 11 through 14 across multiple device viewports.

## Repository Contents
- **[IAP_CHECKLIST.md](./IAP_CHECKLIST.md):** Rapid manual testing checklist with results.
- **[IAP_TEST_CASES.md](./IAP_TEST_CASES.md):** Detailed test cases covering positive, negative, and edge-case scenarios.
- **[FINAL_QA_REPORT_BUILD_VERIFICATION.md](./FINAL_QA_REPORT_BUILD_VERIFICATION.md):** Executive summary of the build status, UI/UX analysis, and known issues.
- **[BUG_REPORT_IAP_001.md](./BUG_REPORT_IAP_001.md) & [BUG_REPORT_IAP_002.md](./BUG_REPORT_IAP_002.md):** Detailed bug reports for critical edge cases.
